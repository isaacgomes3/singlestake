# stake37 — Reals Cruzamento 2 Fatores

Extensão Chrome para **Reals** (mesa **237** · Brazilian Roulette).

URL: https://reals.bet.br/play/liveCassino/237?title=Brazilian+Roulette&provider=pragmatic&gameId=%22237%22&category=live

## Estratégia (idêntica à ICE Cruzamento 2F)

- Pares **3×6** e **2×4** em paralelo — indica no match
- Empate fecha indicação; chave **sem gale** e **sem clique** (observação)
- Gráfico run-up / drawdown; contadores OK/ERR por gatilho

## Build

```bash
npm run extension:2f:sync
npm run extension:reals2f:build
```

Chrome → `chrome://extensions` → Carregar sem compactação → pasta `extension-reals-cruzamento-2f/`
