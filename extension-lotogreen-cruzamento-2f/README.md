# stake37 — Lotogreen Cruzamento 2 Fatores

Extensão Chrome para **Lotogreen** (mesa **225** · Auto Roulette).

URL: https://lotogreen.bet.br/play/pragmatic/auto-roulette

## Estratégia (idêntica à ICE Cruzamento 2F)

- Gatilho **2×4** — indica no match
- Empate fecha indicação; chave **sem gale** e **sem clique** (observação)
- Gráfico run-up / drawdown; contadores OK/ERR

## Build

```bash
npm run extension:2f:sync
npm run extension:loto2f:build
```

Chrome → `chrome://extensions` → Carregar sem compactação → pasta `extension-lotogreen-cruzamento-2f/`
