# stake37 — GoldeBet Cruzamento 2 Fatores

Extensão Chrome para **GoldeBet** (mesa **230** · Roulette 3).

URL: https://goldebet.bet.br/play/pragmatic/roulette-3

## Estratégia (idêntica à ICE Cruzamento 2F)

- Pares **3×6** e **2×4** em paralelo — indica no match
- Empate fecha indicação; chave **sem gale** e **sem clique** (observação)
- Gráfico run-up / drawdown; contadores OK/ERR por gatilho

## Build

```bash
npm run extension:2f:sync
npm run extension:golde2f:build
```

Chrome → `chrome://extensions` → Carregar sem compactação → pasta `extension-goldebet-cruzamento-2f/`
