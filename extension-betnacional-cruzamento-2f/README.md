# stake37 — Bet Nacional Cruzamento 2 Fatores

Extensão Chrome para **Bet Nacional** (mesa **225** · Auto Roulette).

URL: https://betnacional.bet.br/casino/game/225a54

## Estratégia (idêntica à ICE Cruzamento 2F)

- Gatilho **2×4** — indica no match
- Empate fecha indicação; chave **sem gale** e **sem clique** (observação)
- Gráfico run-up / drawdown; contadores OK/ERR

## Build

```bash
npm run extension:2f:sync
npm run extension:bn2f:build
```

Chrome → `chrome://extensions` → Carregar sem compactação → pasta `extension-betnacional-cruzamento-2f/`
