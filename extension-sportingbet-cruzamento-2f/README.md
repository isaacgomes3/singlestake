# stake37 — Sportingbet Cruzamento 2 Fatores

Extensão Chrome para **Sportingbet** (mesa **227** · Roulette 1).

URL: (abrir mesa manualmente)

## Estratégia (idêntica à ICE Cruzamento 2F)

- Gatilho **2×4** — indica no match
- Empate fecha indicação; chave **sem gale** e **sem clique** (observação)
- Gráfico run-up / drawdown; contadores OK/ERR

## Build

```bash
npm run extension:2f:sync
npm run extension:sportingbet2f:build
```

Chrome → `chrome://extensions` → Carregar sem compactação → pasta `extension-sportingbet-cruzamento-2f/`
