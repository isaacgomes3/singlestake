EXTENSÃO CHROME — SINGLESTAKE PLAYTECH v1.4.9
==========================================

Pasta desta extensão:
  C:\Users\PC\Documents\GitHub\singlestake\extension

INSTALAR (1 minuto)
-------------------
1. Chrome → chrome://extensions
2. Modo de programador → LIGADO
3. "Carregar sem compactação"
4. Seleccionar a pasta "extension" acima
5. Fixar o ícone na barra (opcional)

MODOS
-----
DEMO  (badge azul "D") — só destaca o botão na mesa, NÃO aposta
REAL  (badge verde "R") — envia clique real na aposta exterior

Troque no popup da extensão (ícone na barra).

TESTE RÁPIDO
------------
1. Abra uma mesa Playtech no operador (ex. br4.bet.br/play/playtech/...)
2. Clique no ícone da extensão
3. Modo DEMO → botão "Par" ou "Ímpar"
4. Deve aparecer destaque azul no botão da mesa
5. Só passe a REAL quando o alvo estiver correcto

PAINEL PLAYTECH (teu sistema)
-----------------------------
O painel pode enviar sinais via postMessage ou:

  window.__singlestakeExtension.sendSignal({
    betKey: "even",      // odd | even | red | black | low | high
    label: "Par",
    mesaUrl: "https://br4.bet.br/play/playtech/...",
    signalId: "mesa:19:Par:0",
    mode: "demo"         // opcional — senão usa o modo do popup
  });

CÓPIA NO AMBIENTE DE TRABALHO
-----------------------------
  npm run extension:copy-desktop

Depois instale a pasta Desktop\singlestake-extension
