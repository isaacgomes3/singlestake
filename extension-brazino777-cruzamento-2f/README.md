# stake37 — Brazino777 Cruzamento 2 Fatores

Extensão Chrome para **Brazino777** (mesa **237** · Brazilian Roulette).

URL: https://www.brazino777.bet.br/game/brazilian_roulette_pr

## Estratégia (idêntica à ICE Cruzamento 2F)

- Gatilho **2×4** — indica no match
- Empate fecha indicação; chave **sem gale** e **sem clique** (observação)
- Gráfico run-up / drawdown; contadores OK/ERR

## Build

```bash
npm run extension:2f:sync
npm run extension:brazino2f:build
```

Chrome → `chrome://extensions` → Carregar sem compactação → pasta `extension-brazino777-cruzamento-2f/`
