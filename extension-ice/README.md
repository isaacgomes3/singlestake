# stake37 — Extensão ICE

Extensão Chrome **independente** da extensão KTO e br4. Estratégia **2 Fatores · cruzamento sequencial** na **Roulette 2 Extra Time** (mesa **201**) do site [ice.bet.br](https://ice.bet.br).

## Instalação

1. `npm run extension:ice:build`
2. Chrome → `chrome://extensions` → Modo programador
3. «Carregar sem compactação» → seleccione a pasta `extension-ice/`

## Uso

1. Abra [Roulette 2 Extra Time no ICE](https://ice.bet.br/games/tag/roulette/liveroulettea-pragmaticexternal) e aguarde o jogo carregar
2. Calibre as apostas exteriores (📍 no popup) no iframe da roleta
3. Ligue a estratégia no painel flutuante ou no popup da extensão

## Mesa DGA

- **Table ID:** 201
- **URL:** `https://ice.bet.br/games/tag/roulette/liveroulettea-pragmaticexternal`

Mesma lógica da extensão KTO (gales entre gatilhos, 6 s de espera, etc.).
