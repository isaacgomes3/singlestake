# stake37 — Ultra Cruzamento 2 Fatores

Extensão Chrome para **Ultra** (mesa **227** · Roulette 1).

URL: https://ultra.bet.br/jogo/3330041/pragmatic-play-live/roulette-1

## Estratégia (idêntica à ICE Cruzamento 2F)

- Gatilho **2×4** — indica no match
- Empate fecha indicação; chave **sem gale** e **sem clique** (observação)
- Gráfico run-up / drawdown; contadores OK/ERR

## Build

```bash
npm run extension:2f:sync
npm run extension:ultra2f:build
```

Chrome → `chrome://extensions` → Carregar sem compactação → pasta `extension-ultra-cruzamento-2f/`
